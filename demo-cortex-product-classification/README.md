# Glaze & Classify

This project has moved to its own repository:

**https://github.com/sfc-gh-miwhitaker/glaze-and-classify**

Inspired by a real customer question: *"How do I translate and classify my international product catalog using only SQL?"* The project answers that question four ways -- from brittle SQL keywords to Cortex AI pipelines to a custom vision model -- and compares the results side by side.
