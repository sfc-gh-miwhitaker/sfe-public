# Usage Guide

## Fleet Map Tab

- Vehicles shown as colored dots on the Atlanta metro map
- Green = in transit, Yellow = at customer stop, Red = depot/parked
- Blue dots = customer locations
- Click any vehicle for driver name, route, and speed
- Expand "Vehicle Detail Table" for full fleet status

## IoT Dashboard Tab

- Garment status breakdown (IN_SERVICE, LOST, RETIRED)
- RFID event volume by type (CHECK_IN, WASH, DRY, FOLD, DISPATCH, DELIVER, LOST)
- Loss rate by customer and garment type
- Recent events table with timestamp and location

## CFO Chat Tab

- Click a sample question or type your own
- Questions are answered via the SV_IOT_FINANCIAL semantic view
- Results display as data tables
- For full agent capabilities, use Snowflake Intelligence directly

## Snowflake Intelligence

The CFO Assistant agent is registered with Snowflake Intelligence. Open the Intelligence sidebar in Snowsight to chat with it directly.
